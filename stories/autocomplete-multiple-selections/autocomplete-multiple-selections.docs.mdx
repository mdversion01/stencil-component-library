# Autocomplete Multiple Selections

<Description of={AutocompleteMultipleSelectionsStories} />

## HTML

```html
<autocomplete-multiple-selections
  input-id="acms-example"
  label="Favorite fruits"
  placeholder="Type to search"
  editable
  add-btn
  badge-variant="primary"
></autocomplete-multiple-selections>

<script>
  const el = document.querySelector('autocomplete-multiple-selections')

  el.options = [
    'Apple',
    'Banana',
    'Orange',
    'Mango',
    'Blueberry',
    'Strawberry'
  ]

  el.value = ['Apple', 'Mango']

  el.addEventListener('multiSelectChange', (event) => {
    console.log('multiSelectChange:', event.detail)
  })
</script>
```
