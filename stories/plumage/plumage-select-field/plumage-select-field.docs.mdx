# Plumage Select Field

<Description of={PlumageSelectFieldStories} />

## HTML

```html
<plumage-select-field-component
  label="Fruits"
  select-field-id="fruit-example"
  default-option-txt="Select a fruit"
></plumage-select-field-component>

<script>
  const el = document.querySelector('plumage-select-field-component');

  el.options = [
    { value: 'apple', name: 'Apple' },
    { value: 'banana', name: 'Banana' },
    { value: 'cherry', name: 'Cherry' },
  ];

  el.addEventListener('valueChange', (event) => {
    console.log('valueChange:', event.detail);
  });
</script>
```
